SYSTEM DESCRIPTION LOG

Vault‑Tec Archive Terminal — автономный довоенный терминал, разработанный
для извлечения, обработки и архивации видеоматериалов с YouTube посредством
модуля yt-dlp. Система функционирует без административных прав, автоматически
разворачивает Python, FFmpeg, aria2 и необходимые зависимости.

Поддерживается гибридный offline-first режим, обеспечивающий полную
автономность после первичной инициализации. Терминал может быть упакован
в EXE для полевой эксплуатации.

Разработано по стандартам Vault‑Tec Industries:
минимализм, надёжность, автономность, устойчивость к нестабильным сетям.

--------------------------------------------------------------------------

SYSTEM CAPABILITIES

- автоматическая установка Python 3.12 без прав администратора
- гибридная offline-first установка зависимостей
- поддержка высокобитрейтных загрузок через yt-dlp
- интеграция FFmpeg
- ускоренная загрузка фрагментов через aria2
- автономная работа после первого запуска
- интерфейс и логика, соответствующие терминалам Vault‑Tec
- возможность упаковки в EXE

--------------------------------------------------------------------------

INSTALLATION AND EXECUTION PROTOCOL

MODE A — EXECUTABLE LAUNCH (RECOMMENDED)
1. загрузить файл "VaultTecTerminal.exe" из раздела Releases
2. выполнить запуск двойным кликом
3. дождаться автоматической установки компонентов

EXE работает автономно, не требует PowerShell и не запрашивает
