Vault‑Tec Archive Terminal — автономный терминал корпоративного уровня, разработанный для безопасного извлечения и архивации видеоматериалов с YouTube посредством модуля yt-dlp.  
Система функционирует без прав администратора, автоматически разворачивает Python, FFmpeg, aria2 и необходимые зависимости, поддерживает гибридный offline-first режим и может быть упакована в EXE для полевой эксплуатации.

Создано по стандартам довоенной Vault‑Tec Industries:  
минимализм, надёжность, автономность, устойчивость к нестабильным сетевым условиям.

## ВОЗМОЖНОСТИ

- автоматическая установка Python 3.12 без прав администратора  
- гибридная offline-first установка зависимостей  
- поддержка высокобитрейтных загрузок через yt-dlp  
- интеграция FFmpeg  
- ускоренная загрузка фрагментов через aria2  
- полная автономность после первого запуска  
- интерфейс и логика в стиле терминалов Vault‑Tec  
- возможность упаковки в EXE

## УСТАНОВКА И ЗАПУСК

Система поддерживает два режима запуска.

### Вариант 1 — запуск EXE (рекомендуется)

1. загрузить файл `VaultTecTerminal.exe` из раздела Releases  
2. запустить двойным кликом  
3. система автоматически установит необходимые компоненты и перейдёт в рабочий режим  

EXE работает автономно, не требует PowerShell и не запрашивает административных прав.

---

### Вариант 2 — запуск PowerShell-версии

1. загрузить файл `VaultTecTerminal.ps1`  
2. открыть PowerShell  
3. выполнить команду:
powershell -ExecutionPolicy Bypass -File .\VaultTecTerminal.ps1
